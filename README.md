# about

salimtype is a fork of [monkeytype.com](<[monkeytype.com](https://monkeytype.co)>) with quotes extracted from [salim quotes api](https://watasalim.vercel.app/).

# live demo

see the app live here: [live demo](https://salim-type.web.app/)
